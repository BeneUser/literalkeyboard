# literalkeyboard
Do you ever wanted to play the keyboard, but you don't have a keyboard - but you do have a keyboard? Like, literally!

I've just started to work on this project, so let's see how and where this goes. The goal is to be get the most functionality out of your PC or laptop keyboard so you can almost play the piano/other instruments like you would with a real keyboard. Various little tricks are supposed to bridge the "hardware gap", like being able to shift the keyboard so you get the range that you should have. How to do this tricks will be customizable, so you can do it the way you like most. :)
